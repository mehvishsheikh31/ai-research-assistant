# ai-research-assistant

AI Research Paper Intelligence System
A Production-Grade RAG Assistant for researchers and students to ingest PDFs, query document insights with citations, and generate automated literature reviews.

Tech Stack: FastAPI, LlamaIndex, FAISS, Mistral (Ollama), and Reflex UI.

Key Features: Multi-PDF support, citation tracking, and paper summarization.

Goal: Streamline literature reviews through grounded AI generation.
